Análisis Satelital NDVI

Dashboard Analítico

Visualice estadísticas históricas, gráficos temporales y métricas clave de sus parcelas con interfaz intuitiva y reportes automatizados.

Pronóstico Meteorológico

Estrés Hídrico

Detecte zonas con déficit hídrico mediante índice NDMI para optimizar sistemas de riego y conservar recursos.

Mapas Interactivos 3D

API

API REST Potente y Fácil de Usar

Integre datos satelitales y meteorológicos en sus aplicaciones con nuestra API robusta

Endpoints Principales

📍 Gestión de Parcelas

GET /api/parcels/

Obtiene la lista completa de parcelas registradas

Análisis NDVI Histórico

POST /api/parcels/ndvi-historical/

Obtiene análisis histórico de índice vegetativo NDVI

Pronóstico Meteorológico

GET /api/parcels/weather-forecast/{parcel_id}/

Pronóstico meteorológico de 7 días para una parcela

Escenas Satelitales por Rango de Fechas

GET /api/parcels/parcel/{id}/scenes/?start_date=2024-01-01&end_date=2024-12-31

Obtiene escenas satelitales filtradas por fechas

Estrés Hídrico Histórico

POST /api/parcels/water-stress-historical/

Análisis histórico de estrés hídrico mediante NDMI
